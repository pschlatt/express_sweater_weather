# How To:

<details>
           <summary>## Setup </summary>
           <p>- Determine postgres username in CLI with ```psql```

- Update /config/config.json with

|Provided      | Fill With    |
|------------- | -------------
|"dialect":    | "postgres"   |
|"username":   | "<your_username>"|


- $ npx sequelize db:create

- $ npx sequelize db:migrate</p>
</details>





## Run Application Locally

npm start

## Run Tests

npm test
